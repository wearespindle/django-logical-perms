``configs`` module
==================

.. note::
    You can use the config classes to create generic field-based permission configs. These can then be used by
    various API framework integration layers to create field-based API permissions. You can read more about them
    :ref:`here <integrating_api_frameworks>`.

Provides helper classes for defining field-based permission configs.
